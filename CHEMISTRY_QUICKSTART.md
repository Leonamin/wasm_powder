# 🧪 화학 반응 시스템 - 빠른 시작 가이드

## 🎮 테스트 시나리오

### 1. 나무 연소 실험 🔥

**준비물**: 나무, 산소, 불

**실험 순서**:
1. 나무를 화면에 그리기 (🪵 버튼)
2. 나무 주변에 산소 배치 (💨 O₂ 버튼)
3. 불을 나무 근처에 배치 (🔥 버튼)
4. 나무가 200°C 이상으로 가열되면 연소 시작!

**예상 결과**: 나무 → 불로 변환, 산소 소모, 열 방출

---

### 2. 수소 폭발 실험 💥

**준비물**: 수소, 산소, 불

**실험 순서**:
1. 수소와 산소를 섞어서 배치
2. 불을 근처에 배치하여 점화
3. 500°C 이상이 되면 폭발!

**예상 결과**: 
- 강력한 폭발 (반경 5칸)
- 수소 + 산소 → 수증기
- 주변 입자 날아감

**⚠️ 주의**: 매우 강력한 반응입니다!

---

### 3. 물-리튬 반응 실험 ⚡

**준비물**: 물, 리튬

**실험 순서**:
1. 물을 화면에 그리기 (💧 버튼)
2. 리튬을 물에 떨어뜨리기 (⚛️ Li 버튼)
3. 즉시 격렬한 반응 발생!

**예상 결과**:
- 폭발 (반경 4칸)
- 물 → 수소 기체 발생
- 리튬 → 불로 변환

**💡 팁**: 나트륨도 비슷하게 반응하지만 약간 약합니다.

---

### 4. 기름 증발 실험 🌫️

**준비물**: 기름, 불

**실험 순서**:
1. 기름을 화면에 그리기 (🛢️ 버튼)
2. 불을 기름 근처에 배치
3. 기름이 300°C 이상으로 가열되면 유증기로 변환

**예상 결과**:
- 기름 → 유증기 (어두운 갈색 증기)
- 유증기는 위로 상승
- 냉각되면 다시 기름으로 응축

---

### 5. 복합 실험: 연쇄 폭발 🎆

**준비물**: 나무, 기름, 수소, 산소, 리튬, 물

**실험 순서**:
1. 중앙에 수소+산소 혼합물 배치
2. 주변에 나무와 기름 배치
3. 한쪽에 물+리튬 배치
4. 불로 점화
5. 연쇄 반응 관찰!

**예상 결과**:
- 수소 폭발 → 주변 가열
- 나무/기름 연소 시작
- 물-리튬 반응 유발
- 대규모 폭발 연쇄!

## 🎨 창의적인 실험 아이디어

### 화염 방사기
```
[나무][나무][나무]
[산소][산소][산소]
[불]  [불]  [불]
```
→ 지속적인 화염 생성

### 수소 엔진
```
[수소]
[산소]  → [불] → 💥
[수소]
```
→ 주기적인 폭발로 추진력 생성

### 기름 램프
```
    [불]
[기름][기름][기름]
```
→ 기름이 천천히 연소하며 빛 발산

## 🔬 고급 팁

### 온도 관리
- **가열 브러시** (🔥): 선택한 영역을 200°C로 가열
- **냉각 브러시** (❄️): 선택한 영역을 -20°C로 냉각
- 반응 속도를 조절하는 데 유용!

### 브러시 크기 조절
- **마우스 휠**: 브러시 크기 1~20 조절
- 큰 브러시로 빠르게 배치
- 작은 브러시로 정밀 작업

### 렌더 모드
- **물질 타입**: 각 물질의 색상으로 표시
- **온도**: 온도에 따라 색상 변화 (파랑→빨강)

---

## 🐛 문제 해결

### 반응이 일어나지 않아요
1. **온도 확인**: 온도 렌더 모드로 전환하여 확인
2. **물질 접촉**: 물질들이 서로 인접해 있는지 확인
3. **확률**: 반응은 확률적이므로 여러 번 시도

### 폭발이 너무 강해요
1. **소량 사용**: 수소/리튬을 적게 배치
2. **거리 유지**: 중요한 구조물과 거리 두기
3. **벽으로 보호**: 벽(🧱)은 폭발에 강함

### 성능이 느려요
1. **입자 수 줄이기**: 전체 지우기 버튼 활용
2. **브라우저 새로고침**: 메모리 정리
3. **화면 크기**: 작은 영역에서 실험
